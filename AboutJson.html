Học JSON
qua hình ảnh

JSON là "ngôn ngữ chung" giữa bạn và AI. Khi bạn hiểu JSON, bạn có thể kiểm soát chính xác cách AI tạo hình ảnh, video, quiz, và mọi sản phẩm giáo dục.

"key" — Tên trường "string" — Văn bản number — Số true/false — Đúng/Sai 
{
  "teacher":  "Nguyễn Văn An",    ← văn bản
  "subject":  "Toán",
  "grade":    8,                  ← số
  "homeroom": true,               ← đúng/sai
  "classes":  ["8A1", "8A2"]      ← danh sách
}

Phần 1

JSON là gì?

JSON (JavaScript Object Notation) là một "biểu mẫu" chuẩn mà máy tính đọc được hoàn hảo. Giống như tờ khai có ô rõ ràng, JSON có cấu trúc rõ ràng.

Tại sao giáo viên cần biết JSON? Khi bạn dùng AI để tạo hình ảnh, video, quiz games, hoặc xử lý dữ liệu hàng loạt — tất cả đều dùng JSON để giao tiếp. Hiểu JSON = Kiểm soát AI chính xác hơn.

Phần 2

4 Khối Xây Dựng Của JSON

JSON chỉ có 4 kiểu cấu trúc. Nắm được 4 khối này, bạn đọc/viết được mọi file JSON.

1

Key-Value
Cặp Khóa — Giá trị

Mỗi thông tin gồm một tên (key) và một giá trị (value), giống như ô trong biểu mẫu.

{ "name": "Nguyễn Văn An", "age": 35, "active": true }
2

Array
Danh sách

Dùng ngoặc vuông [ ] để chứa danh sách nhiều giá trị cùng loại.

{ "subjects": [ "Toán", "Tin học", "STEM" ] }
3

Nested Object
Đối tượng lồng nhau

Một object chứa bên trong object khác — như thư mục con bên trong thư mục cha.

{ "school": { "name": "THCS Nguyễn Du", "city": "Buôn Ma Thuột" } }
4

Array of Objects
Danh sách đối tượng

Danh sách chứa nhiều object — như bảng điểm với nhiều dòng học sinh.

{ "students": [ {"name":"An", "score":8.5}, {"name":"Bình", "score":7.0} ] }
5 Quy tắc vàng: (1) Key luôn ngoặc kép "key". (2) String luôn ngoặc kép "text". (3) Số và true/false KHÔNG có ngoặc. (4) Phẩy giữa các phần tử, KHÔNG phẩy sau phần tử cuối. (5) Mọi thứ nằm trong { }.

Phần 4

Lỗi Thường Gặp — Tìm & Sửa

6 lỗi phổ biến nhất khi viết JSON. Mỗi ví dụ có phiên bản sai và đúng.

Lỗi 1: Dùng ngoặc đơn thay vì ngoặc kép

✕ Sai
{ 'name': 'An' }
JSON chỉ chấp nhận dấu ngoặc kép ", không chấp nhận ngoặc đơn '.
✓ Đúng
{ "name": "An" }

Lỗi 2: Dấu phẩy thừa sau phần tử cuối

✕ Sai
{
  "name": "An",
  "age": 30,    ← dấu phẩy thừa!
}
Phần tử cuối cùng trong { } hoặc [ ] KHÔNG được có dấu phẩy sau.
✓ Đúng
{
  "name": "An",
  "age": 30     ← không phẩy
}

Lỗi 3: Quên ngoặc kép cho key

✕ Sai
{ name: "An" }
Key (tên trường) PHẢI có ngoặc kép. Đây là quy tắc bắt buộc.
✓ Đúng
{ "name": "An" }

Lỗi 4: Đặt số hoặc boolean trong ngoặc kép

✕ Không nên
{ "grade": "8", "active": "true" }
Số 8 thành văn bản "8", true thành văn bản "true" — AI xử lý khác nhau!
✓ Đúng
{ "grade": 8, "active": true }

Lỗi 5: Thiếu dấu phẩy giữa các phần tử

✕ Sai
{
  "name": "An"   ← thiếu dấu phẩy!
  "age": 30
}
✓ Đúng
{
  "name": "An",  ← có dấu phẩy
  "age": 30
}

Lỗi 6: Ngoặc không khớp

✕ Sai
{
  "items": ["A", "B", "C"
}  ← thiếu ] đóng array!
✓ Đúng
{
  "items": ["A", "B", "C"]
}
Bài tập: Mở jsonlint.com, dán đoạn JSON có lỗi, và sửa cho đến khi thấy "Valid JSON". Lặp lại 3 lần với các lỗi khác nhau.
Phần 5

JSON Cho Prompt Tạo Hình Ảnh AI

Cấu trúc prompt hình ảnh dạng JSON giúp bạn kiểm soát chính xác từng chi tiết — phong cách, bố cục, đối tượng, và kỹ thuật.

json — image prompt 
{
  "image_request": {
    "description": "Educational poster about the solar system
      for Vietnamese elementary students",
    "elements": [
      {
        "object":   "Sun",
        "position": "center",
        "size":     "large",
        "style":    "cute cartoon with smiling face"
      },
      {
        "object":   "Earth",
        "position": "third from center",
        "label":    "Trái Đất"
      }
    ],
    "style": {
      "art_style":       "kawaii cartoon",
      "color_palette":   "bright and cheerful",
      "text_language":   "Vietnamese",
      "target_audience": "children aged 8-10"
    },
    "technical": {
      "aspect_ratio": "16:9",
      "resolution":   "1920x1080"
    }
  }
}

Cách dùng: Copy phần "look" vào mọi prompt tạo ảnh để nhân vật xuất hiện nhất quán. Chỉ thay đổi "poses" cho mỗi cảnh.

JSON Cho Kịch Bản Video AI

Lên kế hoạch từng scene video bằng JSON, sau đó chuyển thành prompt cho Runway, Kling, hoặc Hailuo.

json — video script (2 scenes mẫu) 
{
  "video": {
    "title":    "Vòng Tuần Hoàn Của Nước",
    "subject":  "Khoa học Lớp 5",
    "duration": 30,
    "scenes": [
      {
        "id":       1,
        "time":     "0:00 – 0:08",
        "visual":   "Aerial view of sunny ocean,
          gentle waves, bright blue sky",
        "camera":   "slow pan right",
        "overlay":  "Bước 1: Bốc hơi nước",
        "narration": "Mặt trời chiếu sáng, làm
          nước bốc hơi lên cao."
      },
      {
        "id":       2,
        "time":     "0:08 – 0:16",
        "visual":   "Water vapor rising, forming

        "overlay":  "Bước 2: Ngưng tụ",
        "narration": "Hơi nước bay lên, gặp lạnh
          và ngưng tụ thành mây."
      }
    ]
  }
}

Phần 7

JSON Cho Quiz Games Giáo Dục

Cấu trúc dữ liệu quiz chuẩn — dùng để import vào Kahoot, Quizlet, hoặc tạo game tùy chỉnh.

json — quiz data (3 loại câu hỏi) 
{
  "quiz": {
    "title":   "Lịch Sử VN — Thời kỳ Bắc thuộc",
    "grade":   6,
    "time":    300,
    "questions": [

      // ─── Câu trắc nghiệm ───
      {
        "type":     "multiple_choice",
        "question": "Hai Bà Trưng khởi nghĩa năm nào?",
        "options": [
          {"text": "Năm 40",  "correct": true},
          {"text": "Năm 248", "correct": false},
          {"text": "Năm 542", "correct": false},
          {"text": "Năm 938", "correct": false}
        ],
        "bloom": "remember"
      },

      // ─── Câu đúng/sai ───
      {
        "type":     "true_false",
        "question": "Lý Bí lập nước Vạn Xuân năm 544.",
        "answer":   true,
        "bloom":    "remember"
      },

      // ─── Câu điền chỗ trống ───
      {
        "type":     "fill_blank",
        "question": "Ngô Quyền đánh quân Nam Hán
        trên sông ______ năm 938.",
        "answer":   "Bạch Đằng",
        "accept":   ["Bach Dang", "Bạch đằng"],
        "bloom":    "remember"
      }

    ]
  }
}
Bài tập: Tạo JSON quiz với 5 câu hỏi cho môn của bạn: 2 multiple_choice, 2 true_false, 1 fill_blank. Kiểm tra tại jsonlint.com.
Phần 8

Xây Dựng Prompt Library Cá Nhân

Lưu tất cả prompt templates vào một file JSON — tái sử dụng, chia sẻ với đồng nghiệp, và cải thiện dần theo thời gian.

json — your prompt library 
{
  "library": {
    "teacher": "YOUR NAME",
    "school":  "YOUR SCHOOL",
    "updated": "2025-09-01",

    "templates": [
      {
        "id":   "LP_001",
        "name": "Giáo án chuẩn",
        "prompt": "Tạo kế hoạch bài dạy:
  Môn: {subject}, Lớp: {grade},
  Bài: {lesson}, SGK: {textbook}.
  Theo CT GDPT 2018.",
        "vars": ["subject","grade","lesson","textbook"]
      },
      {
        "id":   "EX_001",
        "name": "Đề kiểm tra",
        "prompt": "Tạo đề kiểm tra {type}
  {subject} lớp {grade}, {duration} phút.
  {mc} TN + {essay} TL. Có đáp án.",
        "vars": ["type","subject","grade","duration","mc","essay"]
      },
      {
        "id":   "IMG_001",
        "name": "Poster giáo dục",
        "prompt": "A {style} educational poster
  about {topic}. {details}.
  For grade {grade}. {colors} palette.",
        "vars": ["style","topic","details","grade","colors"]
      }
    ]
  }
}
Cách dùng với AI: Copy template từ library, paste vào ChatGPT/Claude kèm lệnh: "Điền các biến {subject}, {grade}, {lesson} với: Toán, 8, Phương trình bậc nhất. Rồi tạo output."
Bài tập cuối: Tạo file JSON prompt library cá nhân với ít nhất 3 templates (1 giáo án, 1 đề thi, 1 hình ảnh). Điền {biến} cho 1 bài cụ thể. Kiểm tra tại jsonlint.com.
